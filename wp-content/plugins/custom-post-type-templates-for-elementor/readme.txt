== Description ==

With the help of this plug-in you can link you posts or a custom post type detail pages to a normal Elementor page. You can style that Elementor page and customize the look & feel of that detail page with elements like post title, post image or post content.

Build your own templates with the free version of Elementor.

== Installation ==
* install [Elementor](https://trk.elementor.com/26724)
* install the plugin

== How to use it ==

https://youtu.be/1ix9GNU2BGc

* create a custom post type
* create an Elementor page and add the included elements (post title, post image, post content)
* go to `Elementor - Custom post pages` and define your rules
* open the post
